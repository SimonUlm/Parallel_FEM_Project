Detailed Description

ComBorderNodes manages all corresponding nodes on all communciation borders of the skeleton.

The ComBorderNodes class consists a List of nodes. This list is not seperated into single ComBorder's but is handled as a consecutive list. The seperation into each border is handled by the setter and getter of this class
