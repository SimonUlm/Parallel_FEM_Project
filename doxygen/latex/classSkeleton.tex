\doxysubsection{Detailed Description}
The skeleton is used for the communciation between the distributed processes. 

The \mbox{\hyperlink{classSkeleton}{Skeleton}} class contains a list of the {\itshape Com\+Borders} which contain the information of the corresponding processes and the edge nodes. Additionally Additionally it contains an instance of \mbox{\hyperlink{classSkeleton_1_1ComBorderNodes}{Com\+Border\+Nodes}} which handles the intervening nodes on each comunication border.
