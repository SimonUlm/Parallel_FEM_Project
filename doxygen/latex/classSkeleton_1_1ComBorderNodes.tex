\doxysubsection{Detailed Description}
\mbox{\hyperlink{classSkeleton_1_1ComBorderNodes}{Com\+Border\+Nodes}} manages all corresponding nodes on all communciation borders of the skeleton. 

The \mbox{\hyperlink{classSkeleton_1_1ComBorderNodes}{Com\+Border\+Nodes}} class consists a List of nodes. This list is not seperated into single \mbox{\hyperlink{classSkeleton_1_1ComBorder}{Com\+Border}}\textquotesingle{}s but is handled as a consecutive list. The seperation into each border is handled by the setter and getter of this class
